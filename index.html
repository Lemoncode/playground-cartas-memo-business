Pasos a paso

Primero crearemos la implementación de la función que queramos testear. Te hemos dejado la estructura básica de las funciones de motor, que tiene que devolver y los parámetros que necesita. Ten en cuenta todo lo que hemos aprendido en los diferentes módulo, hacer funciones pequeñas, que hagan una cosa y solo una cosa, separar las funciones de motor de las de ui etc.
El segundo paso será ir creando los test que testen el código de dicha función. Recuerda tener en cuenta que si tu código tiene if/else if etc, ir probando las diferentes posibilidades que tu código puede llegar a tener. También de "espiar" las partes de tu código que sean necesarias.
Una vez tengas la función implementada y los tests realizados. Una manera sencilla de poder probar tus test será entrando en esta web. Aquí podrás ejecutar tus test ejecutando npm run test
